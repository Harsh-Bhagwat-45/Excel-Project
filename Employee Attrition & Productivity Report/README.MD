# Employee Attrition & Workforce Analytics Dashboard (Excel)

# Project Overview
This project analyzes employee attrition and workforce data using Microsoft Excel.
The goal is to identify key factors affecting employee turnover and generate actionable HR insights through an interactive dashboard.

# Business Problem
Employee attrition is a major challenge for organizations. High turnover increases recruitment cost and affects productivity.

# This project answers questions such as:

Which department has the highest attrition?
Which age group leaves the most?
Does overtime increase employee turnover?
Does salary influence attrition?

# Tools Used
Microsoft Excel
Power Query
Power Pivot
DAX Measures
Pivot Tables
Interactive Slicers
Dashboard KPIs
Total Employees
Attrition Count
Attrition Rate
Average Monthly Salary

# Key Insights

1️⃣ Attrition rate is 16%, meaning 1 out of 6 employees leaves the company.

2️⃣ Research & Development and Sales departments show higher attrition.

3️⃣ Employees aged 26–35 leave more frequently.

4️⃣ Employees with lower salary slabs show the highest attrition.

5️⃣ Employees working overtime have higher turnover.
