# 📊 Loan Risk Analysis Dashboard (Excel)

---

## 📌 Project Overview

This project focuses on analyzing loan data to identify **customer risk patterns** and understand factors affecting loan defaults using Microsoft Excel.

The objective is to provide actionable insights that help financial institutions make better lending decisions and reduce default risk.

---

## 🎯 Business Problem

Loan default is a major challenge for banks and financial institutions as it leads to financial losses and impacts overall portfolio performance.

This project aims to answer:

* Which customers are most likely to default?
* How does credit score impact loan risk?
* Does Debt-to-Income (DTI) ratio affect default?
* Which loan categories are more risky?
* Does existing credit behavior influence risk?
* How does property type affect loan safety?

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* Power Query (Data Cleaning & Transformation)
* Power Pivot (Data Modeling)
* DAX Measures
* Pivot Tables & Charts
* Interactive Slicers

---

## 📊 Dashboard KPIs

* Total Customers
* Safe Customers
* Risky Customers
* Total Loan Amount
* Average Interest Rate

---

## 📈 Key Insights

* Customers with **low credit scores** have higher default risk.
* Higher **DTI ratio** significantly increases the probability of default.
* **Low and medium loan categories** show more defaults compared to high-value loans.
* Customers with **no existing credit history** are more likely to default.
* Majority of loan amount is given to **low-income customers**, increasing overall risk.
* **Primary residence loans** are safer, while **investment properties** show higher risk.

## 💡 Conclusion

The analysis shows that **credit score, DTI ratio, income, and credit behavior** are the key drivers of loan risk.

Financial institutions can reduce defaults by improving credit evaluation, limiting high-risk loans, and focusing on financially stable customers.
