📊 Superstore Sales & Profit Performance Dashboard
📌 Project Overview

This project analyzes retail sales data using Microsoft Excel.
The objective is to uncover key insights related to sales, profit, customers, and regions through an interactive dashboard.

🎯 Business Problem

Retail businesses often struggle to track profitability and identify high-performing areas.

This project answers:

Which category generates the highest profit?

Which region performs best in sales?

Who are the top customers?

How do discounts impact profit?

What is the overall sales trend?

🛠 Tools & Technologies Used

Microsoft Excel

Data Cleaning (TRIM, CLEAN)

XLOOKUP / INDEX-MATCH

Pivot Tables & Charts

Conditional Analysis

Interactive Slicers

📊 Dashboard KPIs

Total Sales

Total Profit

Profit Margin (%)

Total Orders

Total Customers

📈 Key Insights

Overall profit margin is around 11–12%, indicating stable performance.

Technology category generates the highest profit.

Furniture category shows low profitability.

West region performs best in both sales and profit.

Some high-sales customers generate losses due to discounts.

Discounted sales increase revenue but reduce profit margins.
